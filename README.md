# AI Text Analyzer

An AI-based web application that analyzes text using NLP techniques.

## Features
- Text Summarization
- Sentiment Analysis
- Keyword Extraction

## Tech Stack
- Python
- FastAPI (Backend)
- Streamlit (Frontend)
- HuggingFace Transformers
- Pydantic
